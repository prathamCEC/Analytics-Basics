# Analytics Basics

A beginner-friendly repository for learning data analysis with Python using Jupyter Notebooks.

## Overview

This repository contains hands-on practice notebooks and mini projects focused on:

- NumPy
- Pandas
- Data cleaning
- Data visualization
- Exploratory data analysis
- Small real-world data projects

## Learning Plan

### Week 2: NumPy + Pandas
- Learn array operations with NumPy
- Work with Series and DataFrames in Pandas
- Perform basic data manipulation and analysis

### Week 3: Data Cleaning + Visualization
- Clean messy datasets
- Handle missing values
- Visualize data using Matplotlib and Seaborn

### Week 4: Mini Projects
Build small projects such as:
- Student results analysis
- IPL data analysis
- Sales dashboard
- Basic business insights project

## Topics Covered

- Python fundamentals for data analysis
- Numerical operations with NumPy
- Data wrangling with Pandas
- Data cleaning techniques
- Exploratory data analysis
- Visualization and storytelling with data

## Tools Used

- Python
- Jupyter Notebook
- NumPy
- Pandas
- Matplotlib
- Seaborn

## Project Ideas

- Student performance analysis
- IPL match statistics dashboard
- Sales trends and insights
- Customer data analysis

## How to Use

1. Clone the repository
2. Open the notebooks in Jupyter
3. Run the cells step by step
4. Practice and build your own projects

## Goal

To build strong foundations in Python-based data analysis and create simple but meaningful data projects.
